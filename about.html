About

All content on this site is original material with me as the author or creator. Where others contributed, they are acknowledged where appropriate.

This site is built with the Jekyll static site generator using the StartBootstrap Clean Blog template by David Miller. The site is hosted by Github Pages.

Icons on this site are provided by Font Awesome.

Original idea for the Bootstrap portfolio was provided by Ray Pierce.

The code for this site can be found on Github.

I develop the site and write content using the Atom editor on Windows 10 with Ubuntu operating from the Windows Subsystem for Linux. I previously used a Macbook Pro and I liked the hardware, but I prefer Windows to MacOS these days. Trust me, 26-year-old me would wonder if I had a stroke while writing this.
